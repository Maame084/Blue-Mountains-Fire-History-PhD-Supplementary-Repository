# ITRAX XRF and Magnetic Susceptibility Data

This folder contains ITRAX XRF geochemical datasets, magnetic susceptibility measurements, processed elemental ratios, principal component analysis (PCA) outputs, and supplementary figures associated with the sediment cores analysed in this thesis.

The datasets support interpretations of sediment composition, minerogenic input, redox conditions, organic matter dynamics, and broader environmental variability across the Upper Blue Mountains peat swamp systems.

Contents include:

* Original ITRAX XRF datasets
* Magnetic susceptibility measurements
* Processed elemental ratios
* PCA loadings, scores, and variance outputs
* Supplementary figures and analytical outputs

These datasets contribute to the multi-proxy reconstruction of Twentieth Century fire regime variability and associated environmental change.
