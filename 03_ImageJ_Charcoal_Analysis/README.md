# ImageJ and Raw Charcoal Quantification Data

This folder contains the original ImageJ-derived charcoal quantification datasets used in the initial measurement and extraction of charcoal particles from sediment samples analysed in this thesis.

The datasets represent the primary stage of charcoal analysis prior to interpolation, CHAR calculations, transformation procedures, and CharAnalysis-based fire-history reconstruction workflows. These files therefore preserve the foundational raw charcoal measurements from which subsequent fire-regime interpretations were developed.

Contents include:

* Raw ImageJ charcoal particle counts
* Charcoal area measurements
* Site-specific charcoal quantification worksheets
* Original supplementary charcoal quantification datasets
* Preliminary charcoal extraction and measurement outputs

These datasets form the primary inputs for subsequent CHARa, CHARno, charcoal transformation, interpolation, and CharAnalysis workflows presented elsewhere in the supplementary repository.
