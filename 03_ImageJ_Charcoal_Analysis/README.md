# ImageJ and Charcoal Analysis Data

This folder contains ImageJ charcoal quantification datasets, CharAnalysis outputs, processed charcoal accumulation datasets, reproducible R workflows, and supplementary analytical outputs associated with the reconstruction of twentieth-century fire regime variability in the Upper Blue Mountains, eastern Australia.

The datasets and scripts support charcoal quantification, CHAR interpolation, fire-episode detection, screening analyses, and the generation of appendix-ready figures presented throughout the thesis.

Contents include:

* Raw ImageJ charcoal quantification datasets
* CharAnalysis screening outputs
* Processed CHARa and CHARno datasets
* Fire-episode summaries and fire-return interval tables
* Reproducible R scripts for cleaning, extraction, and figure generation
* Supplementary appendix figures and workflow manifests
