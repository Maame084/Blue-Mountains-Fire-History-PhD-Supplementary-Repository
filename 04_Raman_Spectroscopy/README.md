# Raman Spectroscopy Data

This folder contains Raman spectroscopy datasets, calibration experiments, reconstructed spectra, peak fitting outputs, chemometric analyses, and supplementary materials associated with the reconstruction of charcoal thermal characteristics in this thesis.

The datasets support Raman-based interpretations of charcoal structural ordering, thermal decomposition, and relative combustion conditions across modern calibration experiments and sediment-derived charcoal samples from the Upper Blue Mountains, eastern Australia.

Contents include:
- Calibration Raman spectra
- Sediment charcoal Raman spectra
- Peak fitting outputs
- Reconstructed spectra
- Chemometric and statistical outputs
- Supplementary figures and analytical datasets
